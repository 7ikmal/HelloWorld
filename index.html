Gaya cepat

Ubah warna aksen di bagian :root CSS. Misal:

Tambahkan halaman lain

Buat file HTML lain (mis. tentang.html) dan tautkan:

<a href="tentang.html">Tentang</a>

FAQ ringkas

Kenapa halaman belum muncul? Pastikan repo publik, Pages diaktifkan, branch benar, dan file bernama index.html di root.

Bisakah pakai domain sendiri? Ya. Tambah file CNAME berisi domainmu dan arahkan DNS ke GitHub Pages.

Mau developer mode? Gunakan Git di laptop untuk commit dan push perubahan; GitHub akan membangun otomatis.
